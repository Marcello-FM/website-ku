💻 Tata Cara Membuat Website Sederhana

  1. Buka teks editor seperti VS Code atau Notepad++.
  2. Buat file index.html dan style.css.
  3. Tulis struktur HTML dasar di index.html.
  4. Tulis gaya tampilan di style.css.
  5. Hubungkan keduanya dengan tag <link rel="stylesheet" href="style.css">.
  6. Buka file index.html di browser untuk melihat hasilnya.
