📘 Mengenal HTML, CSS, JavaScript, dan Tailwind

HTML adalah bahasa dasar untuk membangun struktur halaman web, seperti teks, gambar, dan tautan.

CSS digunakan untuk mempercantik tampilan web: warna, tata letak, dan animasi.

JavaScript membuat website menjadi interaktif dan dinamis — contohnya tombol, efek klik, atau validasi form.

Tailwind CSS adalah framework CSS modern berbasis utility classes, membuat desain cepat dan efisien tanpa menulis CSS dari nol.
